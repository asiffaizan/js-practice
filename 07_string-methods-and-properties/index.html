String Methods and Properties

Before starting string methods and properties we need know that strings are indexed. So basically we can access any single character of a string by doing something like this.

Expression: 'Hello World'[1]

Result: 'e'

There is only one property that exists.

Property

  • length

Methods (Without Argument)

Topics Description
  • toUpperCase()
    		• it's transform letters to Uppercase
  • toLowerCase()
    		• it's transform letters to Lowercase
  • trim()
    		• it's remove all space from start to end
  • trimStart()
    		• it's remove all space from start
  • trimEnd()
    		• it's remove all space from end

    Methods (With Argument)

    Topics Description
  • includes()
    		•  it's checks, is there have any character which we want to find?
  • indexOf()
    		•  it's shows us the first match characters index.
  • replace()
    		•  it's replace a particular thing which we want to replace.
  • replaceAll()
    		•  it's replace all matching characters.
  • concat()
    		•  it's add multiple strings.
  • padStart()
    		•  it's pads a string from the start like '789'.Exp: padStart(12,'*') = (*********789)
  • padEnd()
    		•  it's pads a string from the end like '789'.Exp: padEnd('*',12) = (789*********)
  • charAt()
    		•  it's provides a character of a particular index. Exp: 'Hello Faizan'.charAt(7) = a
  • charCodeAt()
    		•  it's provide a ascii code of a character. Exp: 'Hello'.charCodeAt(2)= 108;which is a ascii code of small l.
  • split()
    		•  it's split a string into an array. Exp: 'Hello Faizan'.split(' ') = ['Hello','Faizan']. basically its remove the argument which one is pass and convert it intu a array, basis on space.
  • slice()
    		•  it's slice a string. Exp: 'Hello Faizan'.slice(6,12) = Faizan
